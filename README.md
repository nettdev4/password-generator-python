# Генератор паролей на Python

Консольная программа для генерации случайных паролей.

## Возможности

- Выбор длины пароля
- Использование специальных символов
- Проверка корректности ввода
- Генерация случайных паролей

## Пример работы

Введите длину пароля: 12

Использовать специальные символы? да

Ваш пароль:
a#B7k@P2m!Q8

## Запуск

Скачайте файл `generator-password.py` и выполните:

```bash
python generator-password.py
```

## Технологии

- Python 3
- random
- os
